# Sentinel-AC Documentation

Welcome to the **Sentinel-AC Documentation**, the official guide for installing, configuring, and managing Sentinel-AC.

This documentation provides everything you need to get started — from setup instructions and permission configurations to advanced features and updates. Whether you’re a developer, server owner, or admin, Sentinel-AC helps you maintain a secure, stable, and well-managed gameplay environment.

> **Note:** This anti-cheat system is **free**.  
> To get access, DM **@clxsx999** on Discord or join the Discord linked in the project.

Sentinel-AC is a **basic anti-cheat system**, ideal for **small or mid-size communities**.  
If you need a more advanced anti-cheat, consider premium solutions such as **FiveGuard, Fini.ac, Antichix**, etc.

---

## Features

| Feature | Description |
|--------|-------------|
| Anti Spectate | Prevents players from entering spectate mode illegally. |
| Anti Night Vision (Job-Based) | Restricts night vision to allowed jobs only. |
| Anti Heat Vision (Job-Based) | Restricts heat vision to approved jobs. |
| Anti Blips (Job-Based) | Prevents players from seeing unauthorized blips on the map. |
| Anti Explosive Bullets | Blocks the use of explosive ammo unless allowed. |
| Anti Weapon Damage Multiplier | Prevents players from modifying weapon damage. |
| Anti Citizen Damage Multiplier | Blocks modification of player damage scaling. |
| Blacklisted Weapon Detection | Detects and blocks unauthorized weapons. |
| Anti TxD Module | Protects against texture dictionary exploits. |
| OCR Module (Optical Character Recognition) | Detects screen-based cheats using OCR analysis. |
| Anti Mass Entity (Global) | Blocks mass spawning of entities to prevent server lag/crashes. |
| Anti Mass Vehicle, Peds, and Objects | Detects and stops mass spawning of vehicles, peds, or objects. |
| Blacklisted Detection (Peds, Vehicles, Objects) | Prevents usage of blacklisted models. |
| Anti Mass Explosion | Blocks large-scale explosion spawning. |
| Anti Silent Explosion | Detects silent explosions meant to bypass normal logs. |
| Blacklisted Explosion Detection | Blocks unauthorized explosion hashes. |
| Advanced Secure Events (Hooked & Isolated Detection) | Protects custom events against hijacking or injection. |
| Anti Player Kill (Server-Sided Detection) | Detects kill events triggered through exploits. |
| Anti Punch Modification (Server-Sided Detection) | Prevents modified melee damage or punch force. |

Don’t expect premium-level protection — **Sentinel-AC is free and basic**, meant to provide essential protection for smaller servers.  
If you need professional-grade security, premium anti-cheats will always perform better.
